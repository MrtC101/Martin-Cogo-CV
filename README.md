# Static CV

Currículum publicado como sitio estático de una sola página.

## Enfoque
- Un único `index.html` en la raíz
- Sin runtime, sin build en producción
- Hosting estático directo

## Origen
Contenido generado con RenderCV, entorno gestionado con uv y estilizado con Tailwind CSS.

## Stack
- RenderCV
- uv
- Tailwind CSS
- HTML

## Criterio
- Rapido
- Minimalista
- Sencillo
